##  What are Image Transformations?

Image transformations are operations that modify the geometry or appearance of an image. These transformations manipulate pixel positions using mathematical operations (matrices) to achieve effects like moving, rotating, resizing, or distorting images.

## üîß Files in This Repository

### 1. `image_cropping.py`
**Purpose:** Extract a specific region of interest (ROI) from an image.

**Logic:**
- Uses NumPy array slicing to select a rectangular region
- Syntax: `cropped = image[y1:y2, x1:x2]`
- No transformation matrix needed - direct pixel selection

**Use case:** Focus on specific objects, remove unwanted areas.

---

### 2. `image_reflection.py`
**Purpose:** Flip/mirror an image horizontally or vertically.

**Logic:**
- Uses 3√ó3 perspective transformation matrix
- **Vertical flip (X-axis):** `[[1, 0, 0], [0, -1, rows], [0, 0, 1]]`
  - Multiplies Y-coordinates by -1, then shifts down
- **Horizontal flip (Y-axis):** `[[-1, 0, cols], [0, 1, 0], [0, 0, 1]]`
  - Multiplies X-coordinates by -1, then shifts right
- Uses `cv.warpPerspective()` to apply transformation

**Use case:** Create mirror effects, correct image orientation.

---

### 3. `image_rotation.py`
**Purpose:** Rotate an image by a specified angle around a center point.

**Logic:**
- Uses `cv.getRotationMatrix2D(center, angle, scale)` to create 2√ó3 rotation matrix
- Matrix combines rotation + optional scaling
- Positive angles rotate counter-clockwise
- Uses `cv.warpAffine()` to apply transformation

**Use case:** Correct tilted images, align objects.

---

### 4. `image_scaling.py`
**Purpose:** Resize an image (enlarge or shrink).

**Logic:**
- Method 1: Specify exact dimensions: `cv.resize(img, (width, height))`
- Method 2: Use scaling factors: `cv.resize(img, None, fx=2, fy=2)`
- **Interpolation methods:**
  - `INTER_AREA`: Best for shrinking (smooths pixels)
  - `INTER_CUBIC`: Best for enlarging (sharp results)
  - `INTER_LINEAR`: Fast, decent quality

**Use case:** Thumbnail generation, image standardization, zoom effects

---

### 5. `image_shear.py`
**Purpose:** Slant/skew an image along X or Y axis.

**Logic:**
- Uses 3√ó3 perspective transformation matrix
- **Horizontal shear:** `[[1, shear_x, 0], [0, 1, 0], [0, 0, 1]]`
  - Formula: `x' = x + shear_x √ó y`
  - Slants image left/right
- **Vertical shear:** `[[1, 0, 0], [shear_y, 1, 0], [0, 0, 1]]`
  - Formula: `y' = shear_y √ó x + y`
  - Slants image up/down
- Requires larger canvas to fit sheared result

**Use case:** Perspective correction, artistic text effects, simulate 3D tilt

---

### 6. `image_translation.py`
**Purpose:** Shift an image horizontally and/or vertically.

**Logic:**
- Uses 2√ó3 affine transformation matrix: `[[1, 0, tx], [0, 1, ty]]`
- `tx`: pixels to shift right (negative = left)
- `ty`: pixels to shift down (negative = up)
- Formula: `x' = x + tx`, `y' = y + ty`
- Uses `cv.warpAffine()` to apply transformation

**Use case:** Reposition objects, create padding, align images

---

## Transformation Matrix Reference

| Transformation | Matrix Type | Key Values |
|----------------|-------------|------------|
| **Translation** | 2√ó3 Affine | `[[1, 0, tx], [0, 1, ty]]` |
| **Rotation** | 2√ó3 Affine | Generated by `getRotationMatrix2D()` |
| **Scaling** | Built-in function | `resize()` with dimensions or factors |
| **Reflection** | 3√ó3 Perspective | Negative scale + shift |
| **Shearing** | 3√ó3 Perspective | Off-diagonal elements |
| **Cropping** | Array slicing | `img[y1:y2, x1:x2]` |

## Key Concepts

### Affine Transformations (2√ó3 matrix)
- Preserves parallel lines
- Includes: translation, rotation, scaling, shearing
- Applied with `cv.warpAffine()`

### Perspective Transformations (3√ó3 matrix)
- Can converge parallel lines (vanishing point)
- More flexible than affine
- Applied with `cv.warpPerspective()`

### Interpolation Methods
- **INTER_NEAREST**: Fastest, lowest quality
- **INTER_LINEAR**: Balanced speed/quality
- **INTER_CUBIC**: Slower, highest quality
- **INTER_AREA**: Best for downsampling


##  Notes

- All transformations preserve image quality when using appropriate interpolation
- BGR to RGB conversion needed for matplotlib display: `cv.cvtColor(img, cv.COLOR_BGR2RGB)`
- Transformation matrices can be combined for complex operations
- Always consider output canvas size to avoid cropping transformed images
